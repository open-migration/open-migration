Features

Everything your data deserves

🌐

Searchable HTML Archive

One self-contained file. Full-text search, dark theme, syntax-highlighted code, filter by source. Open in any browser. Works offline. No server.

🔮

Obsidian Vault

YAML frontmatter, Obsidian callout blocks, wikilinks, auto-generated Index note. Open directly in Obsidian — no configuration needed.

📝

Plain Markdown

One file per conversation, plus an index README. Works with any editor, any static site generator, any version control system.

🔀

Merge Multiple Sources

Combine ChatGPT + Claude + Gemini exports into one unified archive. omigrate merge --inputs a.json b.zip

🖱️

Web UI

Drag-and-drop interface at localhost:7337. No command line required. pip install "open-migration[web]" && omigrate serve

🔒

100% Local

Zero network requests. No accounts. No API keys. No telemetry. Your conversations never leave your machine.

📦

Zero Dependencies

The core CLI has no required dependencies. Pure Python stdlib. Optional rich for a prettier terminal, optional flask for the web UI.

🔌

Extensible

Adding a new connector (source) or exporter (target) is ~150 lines of Python. The graph architecture keeps sources and targets completely independent.

🧪

Tested & Maintained

38 tests across connectors, exporters, and CLI. CI matrix: Python 3.11/3.12 × Ubuntu/Windows/macOS. Active development.

How it works

Three steps to own your data

1

Export from your AI platform

ChatGPT: Settings → Export data
Claude: Settings → Export Data
Gemini: takeout.google.com

2

Run one command

omigrate convert --input export.zip --open
Auto-detects format. No configuration.

3

Use your data

Search everything. Open in Obsidian. Share the HTML file. Back it up. Take it anywhere.
